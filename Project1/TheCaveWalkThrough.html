THE CAVE WALK THROUGH

The Cave 1.0 walk through:

There is also a word document on my site but I thought that having an HTML page would be easier for access.

  1. Press start adventure.
  2. Press Feel free to press help button any time. Save and load any time. adventure.
  3. Use lantern.
  4. West
  5. Take salt. Use salt if you want. If you do, pick up more since it was used up!
  6. North
  7. Use salt
  8. Take grappling hook. Use it if you want.
  9. Take rope
  10. Use rope or grappling hook
  11. south
  12. south
  13. east
  14. east
  15. take plate armor
  16. use plate armor
  17. west
  18. north
  19. take sword. Use sword if you want.
  20. north
  21. use sword
  22. take pick axe. Use it if you want.
  23. east
  24. use pick axe. Use the Glowing Gem if you want.
  25. west
  26. west
  27. use glowing gem
  28. north
  29. You are out!


NOW… If you want to fully experience all the messages and um, things..….


Things to try from starting point. Start a new game before starting each sequence:

  1. Try to wander in the dark. You won’t get far. Best “see the in the dark descriptions” route: west, north, south, east, north, north, south, east, SAVE. South till you make it alive. Reload then west, south, east.
  2. Use the lantern then: Go west, pick up some salt, go north and try north or east. Try to take the grappling hook. Use the salt. Try to get the grappling hook. Use the grappling hook.
  3. Use the lantern then: Go north, try to go west. Go north again, try to take pick axe.
  4. Use the lantern then: Go north then east, save.. now try to go south. If you make it, take plate armor, save. Now try A) go west. B) go back north. Save. Now try going north again. Reload and try north a few times. There is more than one way to die here. Drop the armor, save and try going north a few more times. You might get turned back..
  5. Use the lantern then: Go north, take sword, use the sword, go north, use sword. Now try to take the pick axe….
  6. Use the lantern then: Go north, go north, go west, try to go south. Pick up the rope and use it.
